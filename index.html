#1.快捷操作

ctr+a 全选
ctr+c 复制
ctr+v 粘贴
ctr+x 剪切
ctr+d 复制粘贴
ctr+z 撤销
ctr+shift+z 取消撤销
按住alt不放，点击鼠标出现多个光标

#2.基础语法

##1.注释
代码中不参与编译执行的部分就是注释，注释对于程序的功能来说没有任何影响，主要用来对代码进行解释说明，方便更好的阅读代码
python中单行注释是在一行文字的最开头添加一个#号
“””
多行注释,使用3个双引号或者3个单引号
“””

##2.语句
2.语句
一行只写一条语句,结束后不需要分号;如果一行需要写多条语句,那么多条语句之间用分号隔开。
如果一条语句很长,需要多行显示,那么在换行前加(注意加\时不能破坏数据)

##3.行和缩进
3.行和缩进
缩进:一条语句的开头不能随便写空格和制表符,否则会报IndentationError
行:要求函数和类的声明前后要有两个换行

##4.标识符与关键字
标识符是用来给变量函数或者类命名用的
要求：有字母，数字和下划线组成； 不能数字开头；
关键字（保留字）：python保留下来有自己特殊功能或意义的一些标识符

##5.常见数据及对应的类型
数字（int float）
文本数据（str–字符串）用’’引起来
布尔值（bool） True False
列表（list），字典（dict），元祖（tuple），集合（set），迭代器（iter），生成器，函数（function），自定义的类

##6.输入输出函数
print()
input()

#3.变量

##1.什么是变量
程序中需要通过变量来保存数据

##2.声明变量与赋值
同时声明变量并赋一样的值
a = b = c = 10
同时声明变量并赋不一样的值
x, y = 100, 200

##3.声明变量和变量重新赋值
声明变量，给变量赋值的原理：先开辟合适的空间存储数据，再让变量指向指定的区域；重新给变量赋值的时候，就重新开辟新的存储空间存储新的数据，让变量指向新的内存区域。
